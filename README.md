# Projekt Black

Projekt Black is a Godot 4.6.3 project for a dark 2D pixel-art shooting RPG with metroidvania, looter shooter, hack-and-slash, and extraction elements.

The project is currently in foundation setup. The first playable slice should prove the basic loop: move, aim, shoot, reload, take damage, kill an enemy, and leave a small test area.

## Development Baseline

- Engine: Godot 4.6.3 stable
- Language: GDScript
- Main platform target: PC
- Art direction: pixel art
- Repository: https://github.com/YuiTatsuki1011/projekt-black
