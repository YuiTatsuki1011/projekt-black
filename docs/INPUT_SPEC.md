# Input Spec

この文書は、プレイヤー操作の仕様を整理するためのものです。

## 現在の指定

- 左移動: Aキー
- 右移動: Dキー
- しゃがみ: Sキー
- インタラクト: Wキー
- ジャンプ: Spaceキー
- 照準: マウス
- 発砲: 左クリック
- リロード: Rキー
- 近接攻撃: Fキー
- 緊急ステップ: Shiftキー

## 初期実装案

最初の実装では、以下の操作で進めます。

```text
A: 左移動
D: 右移動
S: しゃがみ
W: インタラクト
Space: ジャンプ
Mouse: 照準
Left Click: 発砲
R: リロード
F: 近接攻撃
Shift: 緊急ステップ
```

動きには最終的に慣性を付ける予定です。ただし、最初は安全な試作として、まず確実に動かすことを優先します。

## 近接攻撃

Fキーで初期武器のナイフを振ります。

- 最大3段のコンボです。
- 1回ごとにスタミナを30消費します。
- 3回振るとスタミナが0になり、オーバーヒートします。
- オーバーヒート中はスタミナが最大まで戻るまで近接攻撃できません。
- 攻撃方向は、A/Dキーを押していればその方向、押していなければマウスカーソルがある左右方向です。

## 緊急ステップ

Shiftキーで短距離ステップします。

- 地上でのみ使用できます。
- A/Dキーを押しながら使うと、その方向へステップします。
- A/Dキーを押していない場合は、マウスカーソルがある左右方向へステップします。
- ステップ中はごく短い無敵時間があります。
- 連続使用を防ぐため、短いクールタイムがあります。
