# Input Spec

この文書は、プレイヤー操作の仕様を整理するためのものです。

## 現在の指定

- 左移動: Aキー
- 右移動: Dキー
- しゃがみ: Sキー
- インタラクト: Wキー
- ジャンプ: Spaceキー
- 照準: マウス
- 発砲: 左クリック
- リロード: Rキー

## 初期実装案

最初の実装では、以下の操作で進めます。

```text
A: 左移動
D: 右移動
S: しゃがみ
W: インタラクト
Space: ジャンプ
Mouse: 照準
Left Click: 発砲
R: リロード
```

動きには最終的に慣性を付ける予定です。ただし、最初は安全な試作として、まず確実に動かすことを優先します。
