# Input Spec

この文書は、プレイヤー操作の仕様を整理するためのものです。

## 現在の指定

- 左右移動: WDキー
- しゃがみ: Sキー
- インタラクト: Wキー
- ジャンプ: Spaceキー
- 照準: マウス
- 発砲: 左クリック
- リロード: Rキー

## 確認が必要な点

左右移動が「WDキー」と指定されていますが、Wキーはインタラクトにも使う指定です。

一般的には、2D横スクロールでは以下のどちらかが自然です。

```text
A / D: 左右移動
W: インタラクト
S: しゃがみ
Space: ジャンプ
```

または:

```text
W / D: 左右移動
S: しゃがみ
別キー: インタラクト
Space: ジャンプ
```

実装前に、正式な入力仕様を決める必要があります。

## 初期実装案

現時点の推奨は以下です。

```text
A: 左移動
D: 右移動
S: しゃがみ
W: インタラクト
Space: ジャンプ
Mouse: 照準
Left Click: 発砲
R: リロード
```

理由は、Wをインタラクトに残しつつ、左右移動を直感的にできるためです。
