# Input Spec

この文書は、プレイヤー操作の仕様を整理するためのものです。

## 現在の指定

- 左移動: Aキー
- 右移動: Dキー
- しゃがみ: Sキー
- インタラクト: Fキー
- ジャンプ: Spaceキー
- 照準: マウス
- 発砲: 左クリック
- サブ武器構え: 右クリックホールド
- サブ武器使用: 右クリックホールド中に左クリック
- リロード: Rキー
- 緊急ステップ: Shiftキー
- インベントリ開閉: Tabキー
- 銃火器切り替え: 1〜4キー
- 将来用補助入力: Qキー / Eキー

## 初期実装案

最初の実装では、以下の操作で進めます。

```text
A: 左移動
D: 右移動
S: しゃがみ
F: インタラクト
Space: ジャンプ
Mouse: 照準
Left Click: 発砲
Right Hold: サブ武器構え
Right Hold + Left Click: サブ武器使用
R: リロード
Shift: 緊急ステップ
Tab: インベントリ開閉
1-4: 銃火器切り替え
Q/E: 将来のスキルや補助武器用
```

動きには最終的に慣性を付ける予定です。ただし、最初は安全な試作として、まず確実に動かすことを優先します。

## 装備

現在の初期装備は、銃火器1番スロットに `basic_pistol`、サブ武器スロットに `field_knife` が入っている扱いです。
インベントリ画面には銃火器1〜4番スロットとサブ武器スロットが表示されます。現在選択中の銃火器スロットは緑の枠で表示されます。
1〜4キーを押すと、対応する銃火器スロットへ切り替わります。スロットごとの現在装弾数は個別に保持されます。
インベントリ内の武器アイテムは、対応する装備枠が空の時だけ右クリックで即装備できます。装備枠が埋まっている時は、右クリックでは交換しません。
武器アイテムは、インベントリから対応する装備枠へドラッグ&ドロップして装備できます。すでに装備している武器がある場合、その武器はインベントリへ戻ります。
装備中アイテムは、装備枠からインベントリグリッドへドラッグ&ドロップして収納できます。装備中アイテムを右クリックした場合も、インベントリに空きがあれば収納します。
インベントリに空きがない場合は `NO SPACE` 警告を表示し、収納を中断します。
装備中アイテムをインベントリ枠外へドラッグ&ドロップすると、そのアイテムをマップ上へ落とします。
インベントリ内アイテム、または装備枠にマウスを乗せると詳細パネルが表示されます。武器の場合は、左に選択中アイテム、右に同種の現装備アイテムを並べ、良い数値は緑、悪い数値は赤で表示します。比較対象がない通常アイテムは左の詳細パネルだけを表示します。

## 銃とリロード

現在の拳銃は、装着中マガジンと薬室で装弾数を持ちます。

- `basic_pistol` と `heavy_pistol` は、インベントリ上では1x1アイテムです。
- `pistol_magazine` は1x1アイテムです。
- `pistol_magazine` は個体ごとに残弾数を持ち、グリッド内では `Mag 6/6` のように表示します。
- Rキーを押すと、インベントリ内の対応マガジンから、弾が最も多いものを選んで交換します。
- 取り外したマガジンはインベントリへ戻ります。インベントリに空きがない場合はマップ上へ落とします。
- インベントリ内に銃弾だけがあっても、対応する装填済みマガジンがなければRキーでリロードできません。
- 現在はマガジンへ弾を詰める操作は未実装です。これは次の段階で追加します。

## 近接攻撃

近接武器はサブ武器として扱います。右クリックを押している間にサブ武器を構え、構え中に左クリックすると装備中の近接武器を振ります。現在の初期サブ武器はナイフです。

- 最大3段のコンボです。
- 構え中は移動速度が通常の約58%まで下がります。
- 1回ごとにスタミナを30消費します。
- 近接攻撃を使うには、最低15以上のスタミナが必要です。
- 現在スタミナが消費量より少なくても、最低必要値を満たしていれば攻撃できます。
- 例: スタミナ20、消費30の場合でも攻撃でき、攻撃後のスタミナは0になります。
- スタミナが0になると、約2秒間0のまま止まり、その後に回復を始めます。
- オーバーヒート中はスタミナが最大まで戻るまで近接攻撃できません。
- 最低必要値を下回っている時はゲージがグレーになり、サブ武器使用入力でゲージが揺れます。
- 攻撃方向は、A/Dキーを押していればその方向、押していなければマウスカーソルがある左右方向です。

## 緊急ステップ

Shiftキーで短距離ステップします。

- 地上でのみ使用できます。
- A/Dキーを押しながら使うと、その方向へステップします。
- A/Dキーを押していない場合は、マウスカーソルがある左右方向へステップします。
- ステップ中はごく短い無敵時間があります。
- 連続使用を防ぐため、短いクールタイムがあります。

## インベントリ

Tabキーでグリッド式インベントリを開閉します。

- 現在は10x8マスの試作グリッドです。
- 開いている間もゲーム進行は止まりません。
- 開いている間はプレイヤーがバッグを漁る状態になり、移動、射撃、リロード、ステップ、サブ武器使用はできません。
- 通常のTabメニューでは、上段にプレイヤーの装備/ステータス、下段にバックパックを表示します。
- インベントリ中は通常の赤い照準を非表示にします。
- 宝箱や倒した敵などのアイテムコンテナを調べた時は、上段にコンテナ、下段にプレイヤーのバックパックを表示します。
- アイテムコンテナはFキーで開きます。通常のTabメニューとは違い、上段が調べているコンテナ、下段が自分のバックパックになります。
- コンテナ内アイテムは、コンテナ内で移動することも、バックパックへドラッグ&ドロップで移すこともできます。
- バックパック内アイテムも、コンテナ側の空きマスへドラッグ&ドロップで移せます。
- TabキーまたはEscキーで閉じます。
- 被ダメージ時はデフォルトで即座に閉じ、攻撃を受けた側の画面端が赤く点滅します。
- 弾薬は `pistol_ammo` として表示されます。
- 拳銃用マガジンは `pistol_magazine` として表示されます。
- アイテム名、短縮名、サイズ、色、スタック数は `resources/items/*.tres` から読みます。
- アイテムはマウスドラッグで移動できます。
- アイテムをドラッグしている間にRキーを押すと、横縦を入れ替えて回転できます。
- 置ける場所は緑、置けない場所は赤で表示されます。
- スタック可能な同一アイテムを重ねると、最大スタック数まで自動でまとめます。
- 最大スタック数を超えた分は、ドラッグ元のスタックに残ります。
- 武器アイテムは装備枠へドラッグ&ドロップして装備できます。
- 対応する装備枠が空の場合だけ、武器アイテムを右クリックで即装備できます。
- アイテムや装備枠にマウスを乗せると、名前、種類、グリッドサイズ、武器性能、現装備との比較結果を表示します。
- アイテムを枠外へドラッグして離すと、そのアイテムをマップ上へ落とします。
- マップ上のドロップアイテムには、近づいてFキーでインタラクトすると拾えます。

## 2026-06-02: Stack Split

- インベントリ、または外部コンテナ内のアイテムを右クリックすると、小さなコンテキストメニューを表示します。
- コンテキストメニューは、メニュー外の何もない場所を右クリックすると閉じます。
- スタック可能アイテムのメニューでは `Split` を選べます。Splitを選ぶと数量入力ダイアログを開き、キーボード入力、`-10`、`-1`、`+1`、`+10` ボタンで分割数を調整できます。
- スタック可能アイテムをCtrl+右クリックすると、メニューを出さずに現在のスタックを半分に分割します。
- スタック可能アイテムをCtrl+左ドラッグすると、現在のスタックから半分だけを持ち上げ、そのまま任意のグリッド、相手側インベントリ、またはマップ上へ置けます。
- 外部コンテナを開いている時、アイテムをShift+右クリックすると相手側インベントリへ即転送します。外部コンテナ側のアイテムならプレイヤーのバックパックへ、プレイヤーのバックパック側のアイテムなら外部コンテナへ送ります。
- 分割された新しいスタックは、元のスタックに近い空きマスへ配置されます。数量指定分割では、指定した数が新しいスタックになります。
- 空きマスがない場合、分割は行われず `NO SPACE` 警告を表示します。
- プレイヤーインベントリ内の武器は、対応する装備枠が空いている場合のみコンテキストメニューに `Equip` を表示します。
