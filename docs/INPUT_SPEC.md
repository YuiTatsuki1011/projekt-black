# Input Spec

この文書は、プレイヤー操作の仕様を整理するためのものです。

## 現在の指定

- 左移動: Aキー
- 右移動: Dキー
- しゃがみ: Sキー
- インタラクト: Fキー
- ジャンプ: Spaceキー
- 照準: マウス
- 発砲: 左クリック
- リロード: Rキー
- 近接攻撃: Eキー
- 緊急ステップ: Shiftキー
- インベントリ開閉: Tabキー

## 初期実装案

最初の実装では、以下の操作で進めます。

```text
A: 左移動
D: 右移動
S: しゃがみ
F: インタラクト
Space: ジャンプ
Mouse: 照準
Left Click: 発砲
R: リロード
E: 近接攻撃
Shift: 緊急ステップ
Tab: インベントリ開閉
```

動きには最終的に慣性を付ける予定です。ただし、最初は安全な試作として、まず確実に動かすことを優先します。

## 装備

現在の初期装備は、拳銃スロットに `basic_pistol`、近接武器スロットに `field_knife` が入っている扱いです。
インベントリ画面には現在装備中の銃と近接武器が表示されます。
インベントリ内の武器アイテムは、右クリックで対応するスロットへ装備できます。

## 近接攻撃

Eキーで装備中の近接武器を振ります。現在の初期武器はナイフです。

- 最大3段のコンボです。
- 1回ごとにスタミナを30消費します。
- 近接攻撃を使うには、最低15以上のスタミナが必要です。
- 現在スタミナが消費量より少なくても、最低必要値を満たしていれば攻撃できます。
- 例: スタミナ20、消費30の場合でも攻撃でき、攻撃後のスタミナは0になります。
- スタミナが0になると、約2秒間0のまま止まり、その後に回復を始めます。
- オーバーヒート中はスタミナが最大まで戻るまで近接攻撃できません。
- 最低必要値を下回っている時はゲージがグレーになり、Eキーを押すとゲージが揺れます。
- 攻撃方向は、A/Dキーを押していればその方向、押していなければマウスカーソルがある左右方向です。

## 緊急ステップ

Shiftキーで短距離ステップします。

- 地上でのみ使用できます。
- A/Dキーを押しながら使うと、その方向へステップします。
- A/Dキーを押していない場合は、マウスカーソルがある左右方向へステップします。
- ステップ中はごく短い無敵時間があります。
- 連続使用を防ぐため、短いクールタイムがあります。

## インベントリ

Tabキーでグリッド式インベントリを開閉します。

- 現在は10x6マスの試作グリッドです。
- 開いている間はゲーム進行を一時停止します。
- 弾薬は `pistol_ammo` として表示されます。
- アイテムはマウスドラッグで移動できます。
- 置ける場所は緑、置けない場所は赤で表示されます。
- 武器アイテムは右クリックで装備できます。
- アイテムを枠外へドラッグして離すと、そのアイテムをマップ上へ落とします。
- マップ上のドロップアイテムには、近づいてFキーでインタラクトすると拾えます。
