# Content Policy

この文書は、ゴア表現や成人向け要素を含む可能性がある素材を、プロジェクト内で安全に扱うための方針です。

## 基本方針

- コード、ファイル名、ディレクトリ名は中立的な名前にする。
- 露骨な説明をコードコメントやドキュメントに残さない。
- ゲーム内設定として、登場キャラクターは全員18歳以上として扱う。
- Codexは成人向け表現の詳細なデザインや描写を担当しない。
- ゴア表現はゲーム内VFXやダメージ演出として記号的に扱う。

## 命名例

使う:

```text
bind_attack
blood_burst_large
damage_debris_a
heavy_hit_reaction
capture_state
```

避ける:

```text
露骨な内容を直接説明する名前
キャラクターの年齢誤認につながる名前
素材内容を必要以上に細かく説明する名前
```

## アセット配置

- 血液や破片系の画像: `assets/vfx/`
- キャラクター画像: `assets/sprites/`
- 効果音: `assets/audio/`
- データ側の名称は、ゲームプレイ上の役割が分かる程度に留める。
